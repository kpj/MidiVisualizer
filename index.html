About

This application allows you to visualize any midi you can upload (upload them using the file dialog in the left saying "Browse").

Try dragging your mouse around the screen or scrolling with your mouse wheel.

You might also want to check one of the following pages for midi files:

  • http://freemidi.org/
  • http://www.cool-midi.com/
  • http://www.free-midi.org/
